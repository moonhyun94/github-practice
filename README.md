# Git Practice

**Quick Start**

0. git repo 생성 및 폴더 생성
1. echo "# github-practice" >> README.md
2. git init
3. git add README.md
4. git commit -m "first commit"
5. git branch -M main
6. git remote add origin git@github.com:moonhyun94/github-practice.git
7. git push -u origin main
